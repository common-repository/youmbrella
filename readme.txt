== Description ==

Youmbrella adds protection to youtube embedded videos making them available only to your subscribers.

Use Youmbrella to generate leads, grow your email list and build your social media through youtube videos.

Here is what Youmbrella can do for you:

•	Make youtube videos available to subscribers only

•	Create landing pages with videos to subscribers only

•	Create playlist with videos to subscribers only

•	Generate list of emails from who played the videos



Youmbrella App
Youmbrella is a standalone application that integrates with WordPress. You must have an account with Youmbrella in order to take advantage of this plugin. [Click here to create your account](https://admin.youmbrella.com/register).

See more at [Youmbrella.com](https://youmbrella.com/).


**Requirements:** PHP5.6+, WordPress 4.0.x+

== Installation ==
1. Install the Youmbrella plugin either via the WordPress.org plugin repository or by uploading the files to your server.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Paste your Youmbrella Key from [Youmbrella.com](https://youmbrella.com/) into the text area and click save!
